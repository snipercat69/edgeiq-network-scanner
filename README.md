# network-scanner

Network reconnaissance Discord bot. Scans IP addresses, CIDR ranges, hostnames, and URLs for open ports and service detection using pure Python sockets (no nmap required). Supports local subnet scanning.

## Topics
python, discord, bot, network, scanner, cybersecurity, nmap, reconnaissance

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/network-scanner.git
cd network-scanner

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 network-scanner.py
```

## About

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
